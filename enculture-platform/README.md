# Enculture Platform

> AI-powered HR intelligence — Culture, Competencies, and 360 Feedback

## What Is Enculture

Enculture is an AI-native HR technology platform with three core products:

| Product | Purpose |
|---------|---------|
| **Culture Intelligence** | Measure, analyze, and shape organizational culture with AI-driven insights |
| **Competency Assessment** | Map employee capabilities against role and organizational needs |
| **Multi-Rater 360** | Comprehensive 360-degree feedback with AI-enhanced pattern analysis |

## Getting Started

### Prerequisites

- **Node.js** (LTS version) — [Download](https://nodejs.org)
- **Git** — [Download](https://git-scm.com)
- **VS Code** — [Download](https://code.visualstudio.com) (recommended editor)

### Clone & Setup

```bash
git clone https://github.com/YOUR_ORG/enculture-platform.git
cd enculture-platform
npm install
```

### Using Claude Code (AI-Assisted Development)

This project includes a `CLAUDE.md` file that gives Claude Code full context about Enculture. To use it:

```bash
cd enculture-platform
claude
```

Claude Code will automatically read `CLAUDE.md` and understand the business context, tech stack, and coding standards.
