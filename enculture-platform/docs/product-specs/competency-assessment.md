# Competency Assessment — Product Spec

> Status: Draft — Needs input from Product Head and Tech Team

## Vision

Replace generic, one-size-fits-all competency assessments with an AI-powered system that adapts to each organization's unique competency framework and provides actionable development insights.

## Target Users

| Persona | Role | Key Need |
|---------|------|----------|
| HR / Talent Team | Assessment administrators | Easy setup, deployment, and reporting |
| L&D Professional | Development planner | Gap analysis and learning recommendations |
| Manager | Team developer | Understand team strengths and development areas |
| Employee | Assessment taker | Fair, relevant assessment with useful feedback |

## Core Features (To Be Detailed)

### Phase 1 — Foundation
- [ ] Competency framework builder (custom per organization)
- [ ] Assessment creation and deployment engine
- [ ] AI-adaptive questioning (difficulty adjusts based on responses)
- [ ] Individual competency scorecards
- [ ] Role-vs-capability gap analysis

### Phase 2 — Intelligence
- [ ] Organization-wide competency heatmaps
- [ ] AI-generated development plans per individual
- [ ] Succession planning support (competency-based)
- [ ] Skills trend analysis across the organization
- [ ] Predictive performance indicators

### Phase 3 — Integration
- [ ] Link to L&D catalog (recommend learning based on gaps)
- [ ] Integration with Culture Intelligence for holistic view
- [ ] External certification and credentialing support
- [ ] API for third-party LMS/HRIS integration

## Open Questions
